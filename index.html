GitHub hosting page

Utilizamos GitHub Pages para alojar esta página estática.

Utilizamos el módulo de NODE gh-pages para subir la página.

Abrimos terminal:

npm init -y Creamos el archivo package.json

npm i gh-pages Instalamos el módulo gh-pages para subir la web

Creamos el archivo .gitignore dónde irán las carpetas que no vamos a subir

En terminal:

git init Incializamos un repositorio

git status Nos muestra en consola los archivos en rojo

git add . Añadimos los archivos al repositorio

git status Nos muestra en consola los archivos en verde

git commit -m "prueba commit_1" Creamos el commit de git

AHORA SUBIMOS EL COMMIT A GitHub

Entramos en GitHub y creamos un repositorio nuevo Publico.

En este caso le llamamos Alojar-html-GitHub

*Para subir los archivos de git al repositiorio

En terminal:

* git remote add origin https://github.com/MMT74/Alojar_html_GitHub.git

* git push -u origin main

Ahora vamos a subir el directorio al gh-pages con el modulo gh-pages

npx gh-pages -d .
